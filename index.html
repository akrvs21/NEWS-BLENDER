1.About

App name: News Blender
Goal and Vision: 21st century is the era of information and technologies. The information is such a valuable and powerful tool that every man in power will use it and manipulate it. There is one statement for that: "Who owns information owns the world". A recent example of these words happened in 2016, the USA president election period, we saw how the manipulated and compromised information were used very accurately, in other words FAKE NEWS, that influenced, to some degree, the election result. In this example, we can see how filtered information can control people's minds. Another thing that modern human needs are to be informed of everything about this person, it can be his professional news, or some hobbies or sport. For example, news that is related to someone's work is very important for a person to be compatible professional in his field, as a programmer I know that technology changes rapidly every day, to keep up to date, the IT man has to regularly watch the news to see some changes, and it can be applied to any field, today world changing so fast, so people need a source of World Wide, breaking and informative news. So with this app users can View and Share the fresh, hot breaking news from all over the world.
Currently, the app provides three types of news:
(1) Bitcoin and blockchain-related news
(2) Apple products related news
(3) Finance and Stock market-related news

In order to issue the network request to a REST API with Retrofit, we need to create an instance using Retrofit.Builder class and configure it with a base URL. In class ApiClient we have BASE_URL, this is the basic URL of our API where we will make a call.



In ApiInterface file, The endpoints are defined inside of an interface using special retrofit annotations to encode details about the parameters and request method.





Here when we requesting a list of news we get a specific news due to our request, in this case about bitcoin, and in the second GET function we are retrieving from all source news when we search by some keyword.
In MainActivity file have enqueue() function, this function asynchronously sends the request and notifies the app with a callback when a response comes back. Since this request is asynchronous, Retrofit handles it on a background thread so that the main UI thread isn't blocked or interfered with.
